# SIMPLE BOT WA

This is a Simple Whatsapp Bot using baileys.

## Ensure Installed
- git
- node
- npm

## Installation
```bash
git clone https://github.com/agusira/botwa
cd botwa
npm install
```
## Running The Script
```bash
node index.js
```

## Create Your Own Features
Open file handler.js located at folder lib.
### Examples
```javascript
switch(m.command){
  case "test": {
    sock.sendMessage(m.chat, { text: "it's working"})
  }
  break;
}
```

## Features

- Support All Running
- Simple The Code
- No Encrypt
